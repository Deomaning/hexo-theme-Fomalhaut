# IDEA 2024.2.1 版本激活

## 激活方式一：激活码（适合内网）

IDEA 2024.2.1 版本破解，适合内网使用。

![IDEA激活配置](/images/Pasted image 20241012171550.png)

这种方式确实可以，但是外网使用一段时间后就会提示重新激活。

## 激活方式二：License Server（适合外网）

所以外网不使用激活码方式激活，最好使用许可认证方式 **License Server** 去激活，简单快捷（**注意：这种方式只有联网才能使用**）。

下面是网址，里面大多数可以使用：

[JetBrains 系列全家桶激活服务器 (每日更新) - License server (asiones.com)](https://jetbrains.asiones.com/)

## 注意事项

- 内网环境推荐使用激活码方式。
- 外网环境推荐使用 License Server 方式，但需要保持联网。
